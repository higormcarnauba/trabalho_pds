# Banco Imobiliário (PDS)

## Instalação

Para compilar o projeto, certifique-se de ter o [Maven](https://maven.apache.org/) e o JDK 21 instalados. Execute o seguinte comando na raiz do projeto:

```bash
mvn clean install
```

## Como Jogar

Após a compilação, você pode iniciar o jogo utilizando o comando abaixo:

```bash
mvn exec:java -Dexec.mainClass="Main"
```

Siga as instruções exibidas no console para escolher o número de jogadores, nomes e realizar as jogadas.
